## 📖 About This Project

This is my **personal portfolio website** built with pure **HTML, CSS, and JavaScript** — no frameworks, no libraries. It showcases who I am, my skills, the services I offer, and how to contact me.

The design features a **dark futuristic aesthetic** with animated floating shapes, glowing purple gradients, smooth scroll animations, and interactive skill bars — all crafted from scratch.

---

## ✨ Features

- 🎨 **Futuristic Dark UI** — Deep purple gradient background with glowing neon accents
- 🌊 **Animated Background** — Floating bubble shapes with continuous loop animation
- 📊 **Animated Skill Bars** — Skill bars animate into view when scrolled into the viewport
- 🧭 **Smooth Navigation** — Pill-style nav links with smooth scroll behavior
- 🃏 **Interactive Cards** — Service and contact cards with hover lift effects
- 📱 **Responsive Layout** — Works across desktop and mobile screens
- ⚡ **Zero Dependencies** — Pure vanilla HTML/CSS/JS, no frameworks required
- 🔠 **Custom Fonts** — Orbitron (headings) + Inter (body) via Google Fonts

## 🚀 Getting Started

Since this is a pure HTML project, no installation or build step is required.

### 1. Clone the repository

```bash
git clone https://github.com/NikhilTheOG/portfolio.git
cd portfolio
```

### 2. Open in browser

Simply double-click `index.html` or open it with your browser:

```bash
# On Linux/Mac
open index.html

# On Windows
start index.html
```

That's it! No servers, no dependencies, no setup needed. ✅
